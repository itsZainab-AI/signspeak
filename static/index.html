00 About this assistant

SignSpeak is a Generative AI-powered accessibility assistant built for the FIFA World Cup 2026 Smart Stadiums challenge, addressing the Accessibility and Multilingual Assistance pillars.

  • Scan & translate — OCR reads any sign or notice, then translates it and generates a plain-language explanation using a locally-hosted LLM (Gemma 2, via Ollama).
  • Ask a question — the same LLM answers venue questions in natural language, drawn from a stadium FAQ knowledge base.
  • 10 languages supported — Arabic, Chinese, English, French, German, Hindi, Japanese, Korean, Portuguese, Spanish.
  • Fully local-first — OCR, translation, and the LLM all run on-device. No data leaves the stadium, and there are no cloud API costs.
